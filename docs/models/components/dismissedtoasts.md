# DismissedToasts

A record of when, under a certain scopeId, a toast was dismissed

## Example Usage

```typescript
let value: DismissedToasts = {
  name: "<value>",
  dismissals: [
    {
      scopeId: "<id>",
      createdAt: 3091.92,
    },
  ],
};
```
