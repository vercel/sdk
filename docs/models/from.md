# From

A set of environments, expressed as explicit slugs, a named preset, or both. At least one of `slugs` or `preset` must be set.


## Supported Types

### `models.From1`

```typescript
const value: models.From1 = {
  slugs: [
    "<value 1>",
  ],
};
```

### `models.From2`

```typescript
const value: models.From2 = {
  preset: "all-custom",
};
```
