# PayloadHistory

History of MFA state changes (enabled/disabled events). Most recent events first.

## Example Usage

```typescript
import { PayloadHistory } from "@vercel/sdk/models/userevent.js";

let value: PayloadHistory = {
  action: "enabled",
  timestamp: 8929.16,
  method: "user_disabled",
  actorId: "<id>",
  actorType: "user",
};
```
