# QueryParamRedirects

Excludes redirect project domains when \"false\". Includes redirect project domains when \"true\" (default).

## Example Usage

```typescript
import { QueryParamRedirects } from "@vercel/sdk/models/getprojectdomainsop.js";

let value: QueryParamRedirects = "false";
```

## Values

```typescript
"true" | "false"
```
