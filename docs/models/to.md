# To

A set of environments, expressed as explicit slugs, a named preset, or both. At least one of `slugs` or `preset` must be set.


## Supported Types

### `models.To1`

```typescript
const value: models.To1 = {
  slugs: [
    "<value 1>",
  ],
};
```

### `models.To2`

```typescript
const value: models.To2 = {
  preset: "all-custom",
};
```
