# BulkRedirects

## Overview

### Available Operations

* [stageRedirects](#stageredirects) - Stages new redirects for a project.
* [getRedirects](#getredirects) - Gets project-level redirects.
* [deleteRedirects](#deleteredirects) - Delete project-level redirects.
* [editRedirect](#editredirect) - Edit a project-level redirect.
* [restoreRedirects](#restoreredirects) - Restore staged project-level redirects to their production version.
* [getVersions](#getversions) - Get the version history for a project's redirects.
* [updateVersion](#updateversion) - Promote a staging version to production or restore a previous production version.

## stageRedirects
