# Connect

## Overview

### Available Operations

* [createConnector](#createconnector) - Create a connector
* [getConnectorToken](#getconnectortoken) - Get a Connect token
* [createConnectorAuthorizationRequest](#createconnectorauthorizationrequest) - Create a Connect authorization request

## createConnector

Create a connector from type-specific configuration and optionally link it to a project during creation.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="createConnector" method="post" path="/v1/connect/connectors" -->
```typescript
import { Vercel } from "@vercel/sdk";

const vercel = new Vercel({
  bearerToken: "<YOUR_BEARER_TOKEN_HERE>",
});

async function run() {
  const result = await vercel.connect.createConnector();

  console.log(result);
}

run();
```

```typescript

  bearerToken: "<YOUR_BEARER_TOKEN_HERE>",
```

```typescript
    connector: "<value>",
```
