# Drains

## Overview

### Available Operations

* [createDrain](#createdrain) - Create a new Drain
* [getDrains](#getdrains) - Retrieve a list of all Drains
* [deleteDrain](#deletedrain) - Delete a drain
* [getDrain](#getdrain) - Find a Drain by id
* [updateDrain](#updatedrain) - Update an existing Drain
* [testDrain](#testdrain) - Validate Drain delivery configuration
