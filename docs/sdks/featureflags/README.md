# FeatureFlags

## Overview

### Available Operations

* [listFlags](#listflags) - List all flags
* [createFlag](#createflag) - Create a flag
* [getFlag](#getflag) - Get a flag
* [updateFlag](#updateflag) - Update a flag
* [deleteFlag](#deleteflag) - Delete a flag
* [listFlagVersions](#listflagversions) - List flag versions
* [getFlagSettings](#getflagsettings) - Get project flag settings
* [updateFlagSettings](#updateflagsettings) - Update project flag settings
* [listTeamFlagSettings](#listteamflagsettings) - List team project flag settings
* [listTeamFlags](#listteamflags) - List all flags for a team
* [createFlagSegment](#createflagsegment) - Create a segment
* [listFlagSegments](#listflagsegments) - List segments
* [getFlagSegment](#getflagsegment) - Get a segment
* [deleteFlagSegment](#deleteflagsegment) - Delete a segment
* [updateFlagSegment](#updateflagsegment) - Update a segment
* [getDeploymentFeatureFlags](#getdeploymentfeatureflags) - Retrieve the feature flags of a deployment
